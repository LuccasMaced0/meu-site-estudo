Parágrafos e quebra linhas

Você pode escrever de qualquer jeito e de qualquer modo utilizando a tags <p> (de cima a baixo ou baixo a cima) que continuara tudo na mesma linha.

Como quebrar uma linha?

Para quebrar uma linha
como esta, basta usar a tag <br>.

Extra

Para sublinhar um frase pode usar a tag <u>
ex: <u>exemplo de linha sublinada<u>
no site aparecera da seguinte forma
exempo de linha sublinhada
