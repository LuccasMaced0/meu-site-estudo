Parágrafos e quebras linhas

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar expecífico
como esse, você pode usar a tag <br>.

Simbolos especiais

Vamos adicionar alguns simbolos especiais: ® © ™ € £ ¥ ¢ ℜ δ Δ ↑ ↑ ↓ ← →

Para Adicionar simbolos espciais como ® ¥ ™ dentro outros como no paragrafo a cima, basta adicionar & segido por seu simbolo.
Ex: & reg; //REG → para o simbolo ® No site aparecera desta forma → ®

Emojis

Vamos adicionar alguns emojis: 😎 🙃 🙄
Para fazer como descrito acima ↑, basta adicionar a tag &#x segido do numero exdecimal.
Ex: &#x 1F60E
No Site aparecera desta forma → 😎

Observações
