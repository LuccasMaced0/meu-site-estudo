Como adicionar favicon site?

Para adicionar um favicon é muito facil.


use a tag <link rel="shortcut icon" href="nomeicone.ico" type="image/x-icon">

e ficara desta forma

Observação

a tag ficou da seguinte maneira no arquivo index.html ↓

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon>

com favicon.ico sendo o nome do icone
